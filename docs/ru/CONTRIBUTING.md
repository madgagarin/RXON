# Как внести свой вклад в RXON

[EN](https://github.com/madgagarin/rxon/blob/main/CONTRIBUTING.md) | [ES](https://github.com/madgagarin/rxon/blob/main/docs/es/CONTRIBUTING.md) | **RU**

Поскольку это общий уровень протокола, изменения здесь влияют на всю экосистему. Пожалуйста, будьте предельно осторожны.

## Установка

1.  Клонируйте репозиторий и перейдите в эту директорию.
2.  Установите в режиме редактирования:
    ```bash
    pip install -e .[dev]
    ```

## Тестирование

Запустите тесты для проверки целостности протокола:
```bash
pytest tests/
```

## Рекомендации

-   **Ломающие изменения**: Любое изменение в `rxon.models` потенциально является ломающим изменением. Проконсультируйтесь с мейнтейнерами перед отправкой.
-   **Зависимости**: Держите ядро свободным от зависимостей. Сетевой код должен находиться в `rxon.transports`.
-   **Типизация**: Статическая типизация строго обязательна для всех моделей протокола.
